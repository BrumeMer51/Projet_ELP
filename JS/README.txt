Avant de lancer le jeu, télécharger les fichier suivants : 
Card.js , Joueur.js , Partie.js , Tour.js , main.js , package.json

Lancer le programme main.js en rentrant dans le terminal : node main.js

Le jeu se lance et vous demande de rentrer le nombre de joueurs : il faut entrer un nombre supérieur ou égal à 2.
Ensuite, il faut rentrer le nom/pseudo de tous les joueurs.

Les joueurs partagent le clavier, quand c'est leur tour ils ont le choix entre arrêter de jouer (en tapant 1) ou piocher une nouvelle carte (en tapant 2).

Durant la partie, un fichier nommé "Partie_de_Flip7.txt" est créé, et à chaque tour les cartes des joueurs ainsi que leur total y est écrit.
A la fin de la partie, le vainqueur est annoncé dans le terminal et dans le fichier.
