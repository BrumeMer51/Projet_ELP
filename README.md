# Projet_ELP

Le code de la fonction principale de notre projet de filtre est Filtre_Image.go
Pour qu'il fonctionne, il faut éditer dans le code le chemin de l'image vers celui de l'image voulu.

Les codes Serveur.go et Client.go sont comme leurs noms l'indique les versions serveur et client de la fonciton de filtre.
Le client demande à l'utilisateur d'entrer le chemin de l'image après execution du client.
